WTHeat

Food recommendation web app

WTHeat은 메뉴 선택을 빠르게 끝내기 위한 음식 추천 웹앱입니다. 사용자가 길게 고민하지 않고 바로 추천 결과를 받을 수 있는 흐름을 목표로 합니다.

핵심 기능

  • 음식 추천 결과 제공
  • 가볍게 접근할 수 있는 웹앱 구조
  • 모바일에서도 바로 사용할 수 있는 화면

운영 상태

현재 서비스는 https://wtheat.zozindev.com/에서 운영합니다. GitHub 저장소는 private이므로 공개 링크로 노출하지 않습니다.
