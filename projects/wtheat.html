WTHeat

Food recommendation web app

WTHeat은 메뉴 선택을 빨리 끝내려고 만든 음식 추천 웹앱이다. 밥 하나 고르는데 에너지 쓰기 싫어서 만든 쪽에 가깝다.

핵심 기능

  • 음식 추천 결과 제공
  • 가볍게 접근할 수 있는 웹앱 구조
  • 모바일에서도 바로 사용할 수 있는 화면

운영 상태

서비스는 https://wtheat.zozindev.com/에서 돈다. GitHub 저장소는 private이라 공개 안 함.
