TEXT_TO_IMG 정적 웹 도구 제작 기록

2026-06-11

TEXT_TO_IMG는 텍스트를 이미지 파일로 만드는 정적 웹 도구다. 사용자가 문구를 입력하고, 스타일을 조정한 뒤 PNG, JPG, WebP 같은 파일로 저장하는 흐름을 목표로 한다. 서버에 텍스트를 저장하지 않고 브라우저에서 바로 처리하는 구조가 핵심이다.

이런 도구는 빠르게 열리고, 설치 없이 바로 쓸 수 있어야 한다. 그래서 초기 구현은 별도 백엔드나 계정 기능 없이 정적 사이트로 유지하는 방향이 적합했다. 사용자가 입력한 값은 브라우저 안에서 처리되고, 결과 파일만 내려받는 방식이면 운영 비용과 개인정보 부담도 줄어든다.

정적 도구로 운영하는 이유

  • 배포와 유지보수가 단순하다.
  • 서버 저장소 없이도 핵심 기능을 제공할 수 있다.
  • 브라우저만 있으면 바로 사용할 수 있다.
  • 루트 도메인에서는 제작 기록과 사용 맥락을 설명할 수 있다.

TEXT_TO_IMG는 `tti.zozindev.com`에서 실제 도구로 운영하고, zozindev.com에는 프로젝트 상세 페이지와 회고 글을 둔다. 이렇게 하면 사용자는 바로 도구로 이동할 수 있고, 검색 엔진은 도구의 목적과 제작 배경을 설명하는 콘텐츠를 읽을 수 있다.

다음 개선 방향은 예시 템플릿과 사용 사례를 늘리는 것이다. 단순히 변환 기능만 있는 것보다 썸네일, 공지 이미지, 짧은 카드 이미지 같은 실제 사용 상황을 보여주면 도구의 가치가 더 명확해진다.
