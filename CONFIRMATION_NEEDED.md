# 확인이 필요한 운영 결정

구현을 멈출 정도는 아니라 기본값으로 갔다. 그래도 운영 전에 보면 좋은 항목이다.

## 1. 블로그 글의 실제 톤

현재 초기 글은 `zozindev.com` 운영과 프로젝트 제작 기록 중심으로 썼다. 더 개인적인 일상 톤을 섞을지, 개발 기록 중심으로 갈지 결정 필요함.

현재 기본값:

- 개발 기록
- 프로젝트 회고
- 운영 노트
- AdSense/SEO 준비 기록

## 2. 프로젝트 상세 페이지의 공개 범위

WTHeat 저장소는 private이라 GitHub 링크는 안 노출했다. 대신 live URL과 제작 기록만 연결했다.

현재 기본값:

- private repository 링크 미노출
- live URL 노출
- 루트 도메인 내부 프로젝트 설명 페이지 노출

## 3. 향후 정적 사이트 생성기 도입 시점

현재는 HTML 파일 직접 관리 방식이다. 글이 20개 이상 쌓이거나 sitemap 수동 관리가 귀찮아지면 Astro나 Eleventy를 다시 보면 된다.

현재 기본값:

- 당장은 no-build 정적 HTML 유지
- 새 의존성 추가 없음
