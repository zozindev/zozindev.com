# 확인이 필요한 운영 결정

구현을 멈출 정도의 결정은 아니어서 기본값으로 진행했지만, 이후 운영 전에 확인하면 좋은 항목입니다.

## 1. 블로그 글의 실제 톤

현재 초기 글은 `zozindev.com` 운영과 프로젝트 제작 기록 중심으로 작성했습니다. 더 개인적인 일상형 블로그 톤을 섞을지, 개발 기록 중심으로 유지할지 결정이 필요합니다.

현재 기본값:

- 개발 기록
- 프로젝트 회고
- 운영 노트
- AdSense/SEO 준비 기록

## 2. 프로젝트 상세 페이지의 공개 범위

WTHeat 저장소는 private이므로 GitHub 링크는 노출하지 않았습니다. 대신 live URL과 제작 기록만 연결했습니다.

현재 기본값:

- private repository 링크 미노출
- live URL 노출
- 루트 도메인 내부 프로젝트 설명 페이지 노출

## 3. 향후 정적 사이트 생성기 도입 시점

현재는 HTML 파일을 직접 관리하는 방식으로 구현했습니다. 글이 20개 이상 쌓이거나 sitemap 수동 관리가 번거로워지면 Astro 또는 Eleventy 도입을 다시 판단하는 것이 좋습니다.

현재 기본값:

- 당장은 no-build 정적 HTML 유지
- 새 의존성 추가 없음
